# <span style="background:#E6F2F3;color:#0F3C44;padding:4px 10px;border-radius:8px;font-weight:900;">이번 주 문제</span>


## <span style="color:#1F5E63;padding:4px 10px;border-radius:8px;font-weight:900;">1번 문제 – 홀짝에 따라 다른 값 반환하기</span>

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">조진행</span>
  - `n`을 매개변수로 받아 if문으로 홀수/짝수 판별
  - `for`문을 사용해 값을 순회하며 `answer`에 누적
  - 홀짝 판별은 `% 2` 연산 사용
  - 증감식에서 `i -= 2`를 사용하여 불필요한 반복을 줄임

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">이수형</span>
  - `if`문으로 홀수/짝수 분기
  - 홀수: `for`문으로 모든 값을 확인하며 조건에 맞는 값만 더함
  - 짝수: 조건을 만족하면 `i * i`를 `answer`에 누적
  - 삼항 연산자(`?:`)를 사용해 홀짝 판별
  - `Math()` 함수를 활용해서 홀짝 판별

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">손지원</span>
  - `if`문으로 홀수/짝수 판별
  - `for`문 내부에서 다시 `if`문을 사용해 조건 확인
  - 후위 연산자를 활용한 반복문 구성
  - 짝수일 경우 `i²`를 계산해 `answer`에 더함
  - 기본 구조에 충실한 구현 방식이었음

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">제갈도원</span>
  - `if` 문 으로 먼저 홀수/짝수 판별
  - 반복문에서 `n` 이하의 값을 전부 순회하면서 다시 조건 검사
  - 홀수일 경우 홀수만 골라 `answer`에 누적
  - 짝수도 동일하게 조건을 확인한 뒤 제곱하여 더함
  - 문제 요구사항을 그대로 코드 흐름으로 옮긴 방식

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">안정연</span>
  - `% 2` 모듈러 연산으로 홀수/짝수 판별
  - 두 경우 모두 반복문 사용
  - 반복문 증감식에서 `i--` 사용
  ```csharp
  for (int i = n; i >= 0; i--)
  ```
  - 역순 반복을 활용

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">홍정옥</span>
  - 반복문을 먼저 구성한 뒤, 내부에서 조건문 사용
  - `Math()` 함수 활용을 고려했으나
    형변환 과정이 복잡해 조건문 + `i * i` 방식으로 변경
  - 반복문으로 값을 나열한 뒤 논리 연산으로 해결

<br>

## <span style="color:#1F5E63;padding:4px 10px;border-radius:8px;font-weight:900;">2번 문제 - 배열에서 문자열 대소문자 변환하기</span>

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">조진행</span>
  - `for`문으로 배열을 먼저 순회
  - 인덱스의 짝수/홀수를 판별
  - 짝수/홀수에 따라
    `ToLower()`, `ToUpper()` 함수 사용

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">이수형</span>
  - 배열의 길이를 기준으로 반복문 구성
  - 조건문으로 인덱스의 짝수/홀수 구분
  - 삼항 연산자(`?:`)를 사용해 대소문자 변환
  - `ToLower()`, `ToUpper()` 함수 활용
  - 코드 길이를 줄이려는 의도가 보였음

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">손지원</span>
  - `for`문으로 배열의 모든 요소 확인
  - `if`문을 사용해 짝수/홀수 판별
  - 짝수 → `ToLower()`
    홀수 → `ToUpper()`

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">제갈도원</span>
  - `for`문으로 배열을 순회
  - `if`문으로 인덱스의 홀수/짝수 판별
  - 짝수 인덱스는 `ToLower()`
    홀수 인덱스는 `ToUpper()` 적용

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">안정연</span>
  - 문자열을 바로 변환하기보다, 문자(`char`) 단위로 분해하여 처리
  - `char` ,`int` 끼리 변환을 통해 ASCII 코드에 직접 접근하는 방식으로 풀이
  - `ToLower()`, `ToUpper()` 함수의 존재를 몰라
    아스키 코드 값을 직접 계산하여 대소문자 변환을 시도함

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">홍정옥</span>
  - 문자열의 대소문자 변환 개념 자체는 이해하고 있었음
  - 다만, 이를 코드로 구현하는 과정에서 어려움을 겪음

## <span style="color:#1F5E63;padding:4px 10px;border-radius:8px;font-weight:900;">인상 깊었던 아이디어</span>

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">1번 문제</span>
  - 반복문을 먼저 구성한 뒤, 내부에서 조건문 사용

- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">2번 문제</span>
  - `char` 배열과 `int` 배열을 각각 선언
  - 문자열을 문자 하나씩 순회
  - `char → int` 변환으로 ASCII 코드 값 획득
  - 소문자인 경우
    ```csharp
    convertCha[j] -=0x20;
    ```
    와 같이 16진수 값(0x20)을 이용해 대문자로 변환
  - 다시 `int → char` 형변환을 통해 문자열로 복원


- <span style="color:#4F7F83;padding:2px 8px;border-radius:6px;font-weight:800;">3번 문제</span>
  - 직원이 언제 출근할건지 판단
    일주일 동안 주말 뺴고 지각이 없으면 상을 받음
	그 해당하는 직원에 상을 주면 됨
    
    | 단계 | 수행 내용 |
    | --- | --- |
    | 1단계 : 직원 순회 | 모든 직원을 한 명씩 확인 (`for i < schedules.Length`) |
    | 2단계 : 요일 판별 | 현재 확인하는 날짜가 평일인지 주말인지 확인 |
    | 3단계 : 시간 변환 | `HH:MM` 형식을 계산하기 편하게 '총 분'으로 변환 |
    | 4단계 : 지각 체크 | `출근시간(분) - 희망시간(분) > 10` 이면 해당 직원 탈락 |
    | 5단계 : 결과 합산 | 일주일(평일 기준)을 모두 통과한 직원만 `answer++` |
