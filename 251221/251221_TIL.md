# TIL: C# 문자열 반복과 제곱수 확인

**날짜: 2025-12-21 일요일**

## 문자열 각 문자 반복

- 문제: 문자열 `my_string`의 각 문자를 `n`번 반복해서 새로운 문자열 생성
- 배운 점:
  - 배열을 쓸 필요 없음, 빈 문자열 `answer`에 문자 누적 가능
  - 구조:
    ```csharp
    answer = ""
    for 각 문자 c in my_string:
        반복 n번:
            answer에 c 더하기
    ```
  - C# 예시 흐름:
    ```csharp
    string answer = "";
    for (int i = 0; i < my_string.Length; i++)
    {
        for (int j = 0; j < n; j++)
        {
            answer += my_string[i];
        }
    }
    ```
## 제곱수 확인

- C#에서 제곱 연산자를 직접 쓰는 `^`는 거듭 제곱이 아님   
- 배운 범위 내에서는 `Math.Pow` 없이도 가능:
  - `b`의 제곱 = `b * b`
  - `a`가 `b`의 제곱인지 확인: `a == b * b`

## 제곱수 판별 후 조건 반환
- 흐름:
    ```csharp
    x = 1
    while (x * x <= n)
        if (x * x == n)
            return 1
        x++
    return 2
    ```
- 포인트:
  - 제곱이면 즉시 1 반환
  - 반복문 끝나고도 찾지 못하면 2 반환
- 문제 예시:
  - `144 → 1`
  - `976 → 2`

## 배운 점 요약
- 문자열 반복은 빈 문자열에 += 방식으로 충분
- 제곱수 확인은 곱셈으로 비교 가능
- 반복문 안에서 조건 만족 시 즉시 반환하면 코드 간단
